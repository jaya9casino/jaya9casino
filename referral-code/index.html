Find Your Referral Code

Your Jaya9 referral code is usually available inside the account area after login. It may appear as a code, invite link or referral panel depending on the current interface. Copy it directly from the account rather than typing it from memory.

If the code is not visible, check whether referral campaigns are currently active for your account. Some promotions are limited by region, account status or campaign period.

How Referrals Work

A referral code connects a new user’s registration to the existing player who shared the invite. Rewards are normally released only when the invited user meets the campaign conditions, such as registering correctly, verifying the account or completing a qualifying activity.

  1. Log in to your Jaya9 account.
  2. Open the referral or invite section.
  3. Copy the official code or link.
  4. Send it to a friend who is eligible to register.
  5. Track the status from your account panel.

Reward Rules

Referral rewards are not automatic in every case. The invited user may need to be new to the platform, register through the correct link, complete phone verification and meet the minimum activity stated in the promotion rules.

Avoid Common Mistakes

Do not share old screenshots, unofficial shortened links or edited codes. Ask the invited user to register through the exact referral path and avoid creating multiple accounts. If the reward does not track, collect details before contacting support.

  • Share the code exactly as shown in your account.
  • Check that the campaign is still active.
  • Do not promise rewards before the rules are completed.
  • Contact support with the invited username only through official channels.
